DE:

Baue dieses Muster auf:

 C T
 C M
   R
   X

C = Univaersal cable
T = Tesseract
M = Advanced Miner
R = Redstonefackel (Stiel auf dem Boden)
X = Roboter (schaut richtung Miner)

Der Tesseract muss Energie empfangen und Items senden
Der Advanced Miner hat NUR einer Scannercard(OV) drin
Der Roboterer braucht:
    Upgrades:
        Chunckloader
        Inventar
        Inventarcontroller
        Batterie(optional)
    Hardware:
        (CPU + Grafickarte) oder APU
        RAM
        Tastatur
        Bildschirm
        Harddrive (mit OpenOS)
            oder Diskdrive zum installieren von OpenOS
        EEPROM (Lua BIOS)
    Ausrüstung:
        Electric Wrench (Lossless mode) equipped
        Configurator (Wrench mode) 1. Slot im Inventar

EN:

Build this pattern:

 C T
 C M
   R
   X

C = Univaersal cable
T = Tesseract
M = Advanced Miner
R = Redstonetorch (Torch stick on the ground)
X = Roboter (looking to the miner)

The Tesseract has to receive energy and send items
The Advanced Miner ONLY has a scanner card (OV) in it
The robotor needs:
    Upgrades:
        Chunckloader
        Inventory
        Inventory Controller
        Battery(optional)
    Hardware:
        (CPU + Grafickarte) oder APU
        RAM
        Keyboard
        Screen
        Harddrive (with OpenOS)
            or Diskdrive for installing OpenOS
        EEPROM (Lua BIOS)
    Ausrüstung:

        Configurator (Wrench mode) 1. inventory slot
